The New Zealand Puzzle is a laying puzzle created by Philos-Spiele. This web app serves as a virtual implementation of the original, physical puzzle.

Also featured are two variations of the game board arranged by Dr. Keith Brandt and I. These variations are intended to be even more difficult than the original game board.
